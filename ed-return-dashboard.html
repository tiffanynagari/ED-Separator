How to refresh this dashboard

This page never talks to Google APIs directly — it only reads dashboard-data.json sitting next to it in this repo. To update it:

1. Run ed_return_extractor.py in Google Colab (writes the branch + combined Sheets to Drive, same as before, and downloads dashboard-data.json).
2. Copy that file into this repo, replacing the old one:
cp ~/Downloads/dashboard-data.json /path/to/ED-Separator/dashboard-data.json
3. Commit and push:
git add dashboard-data.json && git commit -m "Update data" && git push
4. Reload this page (GitHub Pages updates within ~1 minute of the push).
